# Data Gap Initiative in Africa

## Overview
This project explores data gaps in key sectors of Africa, including agriculture and health, as part of the Data Gap Initiative. The goal is to identify critical areas where data is lacking and provide a comprehensive dataset for analysis.

## Project Components
The repository contains the following documents:

1. **Code**: Scripts used for data collection, cleaning, and analysis.
2. **Dataset**: A curated dataset that highlights the explored data gaps.
3. **Insight Presentation**: A presentation summarizing key findings and insights from the analysis.
4. **Literature Review**: A document reviewing existing literature related to data gaps in the selected sectors.
